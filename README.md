# WeatherNow
Created a modern weather detection website that provides real-time weather data.
It is a **responsive web application** that allows users to search for real-time weather data for any city using the **OpenWeatherMap API**.

# Tech Stack Used
- **Frontend:** HTML, CSS, JavaScript
- **API:** OpenWeatherMap API
- **Deployment:** Vercel
  
# How it works?
1. User enters a city name
2. App fetches live weather data using OpenWeatherMap API
3. Weather details are displayed instantly
4. Users can also ask common weather-related questions
