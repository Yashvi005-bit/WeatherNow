# WeatherNow
Created a modern weather detection website that provides real-time weather data.
It is a **responsive web application** that allows users to search for real-time weather data for any city using the **OpenWeatherMap API**.

# Features 
- Live weather search : Access live weather data updated in real-time
- City-based forecast : Search for any city to get **current weather conditions** including temperature, humidity, and weather description.
- Clean UI with animations

- **Frontend:** HTML, CSS, JavaScript
- **API:** OpenWeatherMap API
- **Deployment:** Vercel
